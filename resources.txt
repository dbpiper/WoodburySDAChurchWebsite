  Item Data:
    Item Data -- API:
      https://pathofexile.gamepedia.com/Path_of_Exile_Wiki:Data_query_API

    Official Item Data (no API):
      -- can be scrapped periodically to check that the data from the Wiki
         is accurate, however this will be slow and potentially buggy so should
         not be relied upon for production.
      https://www.pathofexile.com/item-data

  Stash Tab/Trading Data -- API:
    https://www.pathofexile.com/developer/docs/api-resource-public-stash-tabs

    A bit of description about this API:
      https://pathofexile.gamepedia.com/Public_stash_tab_API

  List of Leagues -- API:
    https://www.pathofexile.com/developer/docs/api-resource-leagues

  Character Inventory -- API:
    https://pathofexile.com/character-window/get-items?character={CharacterName}&accountName={AccountName}

  Get Characters -- API:
    https://www.pathofexile.com/character-window/get-characters?accountName={AccountName}

  Get Passive Skill Tree -- API
    https://www.pathofexile.com/character-window/get-passive-skills?accountName={AccountName}&character={CharacterName}

  Scraping Item Data (use PyPoE) see: https://pathofexile.gamepedia.com/Path_of_Exile_Wiki:To-do_list/3.3.0

  Get Passive Tree Data (needed to figure out what nodes correspond to which hashes from get-passive-skills) -- API/HTML with JSON
    https://www.pathofexile.com/passive-skill-tree

    Then parse the html, looking for the JavaScript variable "passiveSkillTreeData".
